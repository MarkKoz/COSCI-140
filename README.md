### Tasks
Write a custom string class.

### Requirements
* Use the given files `myStringClass.h` and `main.cpp`.
    * Define all functions declared in the class `myStringClass`.
* Comments.
* Meaningful or mnemonic variable names.

### Prohibitions
* Global variables.
* Infinite loops. Examples:
    * `for(;;)`
    * `while(1)`
    * `while(true)`
    * `do { // code } while(1);`
* `break` to exit loops.
* Code found on the internet.
* `string` class from the C++ Standard Library
    * String literals are allowed.
